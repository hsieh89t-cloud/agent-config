# Markdown語法完整指南

Markdown是一種輕量級標記語言，廣泛用於文檔撰寫和筆記記錄。

## 基本語法

### 標題
```
# 一級標題
## 二級標題
### 三級標題
```

### 文字格式
- **粗體**：`**粗體**` 或 `__粗體__`
- *斜體*：`*斜體*` 或 `_斜體_`
- ~~刪除線~~：`~~刪除線~~`
- `行內代碼`：`` `代碼` ``

### 列表
#### 無序列表
```
- 項目一
- 項目二
  - 子項目
```

#### 有序列表
```
1. 第一項
2. 第二項
   1. 子項目
```

### 連結和圖片
- 連結：`[文字](URL)`
- 圖片：`![替代文字](圖片URL)`

### 引用
```
> 這是一段引用
> 可以多行
```

### 代碼區塊
```python
def hello():
    print("Hello, Markdown!")
```

### 表格
```
| 標題1 | 標題2 | 標題3 |
|-------|-------|-------|
| 內容1 | 內容2 | 內容3 |
| 內容4 | 內容5 | 內容6 |
```

## 擴展語法（CommonMark）
- 任務列表：`- [ ] 未完成`、`- [x] 已完成`
- 腳註：`文字[^1]`，然後 `[^1]: 註釋`
- 定義列表
- 表情符號：`:smile:`

## 最佳實踐
1. 每行不超過80字符
2. 適當使用空行分隔段落
3. 為圖片提供有意義的替代文字
4. 使用相對路徑而非絕對路徑
